## How the linking works

- Each chapter file opens with an **"In this chapter"** mini-index (subsection name → page anchor), built from the manual's own running headers.
- Every `Torque:` value found anywhere in the manual is logged in [09-torque-specs.md](09-torque-specs.md) with a link back to its exact source page. The source line itself carries a matching back-link comment.
- SST (special service tool) part numbers are listed at the start of each system chapter and in `01-preparation.md` (not yet cross-linked — say the word if you want that pass too).

## OCR cleanup pass

All 8 chapters have been through a manual OCR-cleanup pass: junk text deleted (reversed strings, diagram-image soup, watermark remnants), cosmetic header spacing fixed. **No numeric value was ever altered or deleted anywhere** — torque, clearance, resistance, voltage, part numbers all stayed byte-identical to the OCR source, even when suspected wrong. Anything ambiguous was left in place and logged in [10-needs-review.md](10-needs-review.md) instead of being guessed at — check that file, especially its "priority items" section, before relying on a spec from this manual.
