# Steering

*Section code: `SR` | source pages 772–796 (manual pages SR-1 to SR-25)*

Covers the manual (non-power) steering system of the AW11 MR2: on-vehicle inspection,
the tilt steering column assembly, and the rack-and-pinion steering gear housing.

Torque is given as **kg-cm (ft-lb, N·m)**, exactly as printed in the manual.

> **PRECAUTIONS**
>
> Care must be taken to replace parts properly because they could affect the performance
> of the steering system and result in a driving hazard.

## On-Vehicle Inspection

<a id="p774"></a>
### 1. Check that steering wheel freeplay is correct

With the vehicle stopped and pointed straight ahead, rock the steering wheel gently back
and forth with light finger pressure. Freeplay should not exceed the maximum limit.

**Maximum freeplay: 30 mm (1.18 in.)**

If excessive, repair.

<a id="p774"></a>
### 2. Check steering linkage and gear housing

1. Check the steering linkage for looseness or damage. Also check that:
   - The tie rod ends do not have excessive play.
   - The boots are not damaged.
   - The boot clamps are not loose.
2. Check gear housing for grease leakage or oozing.

## Steering Column Assembly

<a id="p774"></a>
### Components

*(Figure: exploded view of the tilt steering column assembly — see page image p.777, manual page SR-6.)*

Tightening torques called out on the components and removal diagrams (manual pages SR-3 and SR-6):

| Fastener | Torque — kg-cm (ft-lb, N·m) |
| --- | --- |
| Steering wheel set nut | 350 (25, 34) |
| Universal joint bolts | 360 (26, 35) |
| Tilt steering support / breakaway bracket nuts | 260 (19, 25) |
| Pedal bracket stay bolt | 185 (13, 18) |
| Tilt lever set bolt (LH threads) | 340 (25, 33) |
| Column upper bracket bolts | 195 (14, 19) |
| Main shaft-to-column tube bolts | 130 (9, 13) |
| Tilt steering support nuts | 120 (9, 12) |
| Breakaway bracket adjusting nut (LH threads) | 105 (8, 10) |

Non-reusable parts (per components diagram): the two O-rings.

<a id="p774"></a>
### Removal of steering column assembly

1. Disconnect the negative cable from the battery.
2. Remove the universal joint.
   1. Place matchmarks on the main shaft and joint yoke.
   2. Place matchmarks on the joint yoke and pinion shaft.
   3. Remove the two bolts.
   4. First pull the universal joint from the gear housing, and then pull it out from the main shaft.
3. Remove the steering wheel.
   1. Remove the screw and disconnect the horn connector.
   2. Pull up and remove the wheel pad.
   3. Remove the steering wheel nut.
   4. Place matchmarks on the main shaft and steering wheel.
   5. Using SST, remove the steering wheel. **SST 09609-20011**
4. Remove the following parts:
   1. Scuff plate
   2. Cowl side panel LH
   3. Instrument lower finish panel
   4. Air duct
   5. Column lower cover
5. Disconnect the ignition switch connector and combination switch connector.
6. Remove the pedal bracket stay bolt.
7. Remove the steering column assembly with column upper cover.
   Remove the six nuts and steering column assembly.

<a id="p778"></a>
### Disassembly of steering column assembly

1. Remove the combination switch.
   1. Remove the clamp.
   2. Remove the two screws from the connector.
   3. Remove the four screws and combination switch.
2. Remove the tension springs and grommets.
   1. Pull down the tension springs, and remove the grommets.
   2. Remove the bolts and springs.
3. Remove the breakaway bracket.
   1. Remove the set bolt and tilt lever.
      > **NOTE:** The bolt has LH threads.
   2. Remove the adjusting nut and washer, and pull out the lock bolt.
   3. Remove the two screws and breakaway bracket.
4. Remove the tilt steering support.
   1. Remove the two nuts and the pawl set bolts.
   2. Remove the bushings, O-rings and tilt steering support.
5. Remove the upper bracket from the main shaft.
   1. Remove the three screws and retainer from the upper bracket.
   2. Using snap ring pliers, remove the snap ring.
   3. Release the steering lock.
   4. Loosen the broken-down bolt by tapping the chisel.
   5. Remove the three bolts, and separate the upper bracket and column tube.
6. Remove the main shaft from the column tube.
   1. Remove the two bolts.
   2. Pull out the main shaft from the column tube.
7. Remove the thrust stopper assembly.
   1. Using snap ring pliers, remove the snap ring.
   2. Remove the thrust stopper assembly.

<a id="p780"></a>
### Inspection and repair of steering column assembly

1. Inspect the upper bracket.
   1. Check that the steering lock mechanism operates properly.
   2. Check the upper bearing rotation condition and check for abnormal noise.
2. If necessary, replace the upper bearing.
   1. Using SST, tap out the bearing. **SST 09620-30010 (09631-00020, 09627-30010)**
   2. Pack a new bearing with MP grease.
   3. Using SST, tap in the new bearing. **SST 09620-30010 (09631-00020, 09625-30010)**
3. Inspect the lower bearing.
   Check the bearing rotation condition and check for abnormal noise. If faulty, replace
   the thrust stopper assembly.

<a id="p781"></a>
### Assembly of steering column assembly

*(See Components, manual page SR-6.)*

1. Install the thrust stopper assembly.
   1. Install the thrust stopper assembly.
   2. Using snap ring pliers, install the snap ring.
2. Install the main shaft to the column tube.
   1. Apply molybdenum disulphide lithium base grease to the thrust stopper.
   2. Install the main shaft and torque the bolts.
      **Torque: 130 kg-cm (9 ft-lb, 13 N·m)**
3. Install the upper bracket to the main shaft.
   1. Release the steering lock.
   2. Install the upper bracket.
   3. Torque the three bolts.
      **Torque: 195 kg-cm (14 ft-lb, 19 N·m)**
   4. Using snap ring pliers, install the snap ring.
   5. Install the retainer to the upper bracket with the three screws.
4. Install the tilt steering support.
   1. Apply molybdenum disulphide lithium base grease to the bushings and O-rings, and
      install the two bushings to the column tube.
   2. Install the tilt steering support and pawl set bolts with bushings and O-rings.
      > **NOTE:** Be careful not to damage the bushing and O-ring.
   3. Torque the nuts.
      **Torque: 120 kg-cm (9 ft-lb, 12 N·m)**
5. Install the breakaway bracket.
   1. Apply molybdenum disulphide lithium base grease to the portions indicated by the arrows.
   2. Install the breakaway bracket, two screws and lock bolt.
   3. Install the washer and adjusting nut, and tighten the nut by the tilt lever.
      **Torque: 105 kg-cm (8 ft-lb, 10 N·m)**
      > **NOTE:** The adjusting nut has LH threads.
   4. Adjust the lock bolt so that it is positioned 2/3 of the way up the oval hole on the
      bracket. Then align the top surface of the bracket with the surface of the column in parallel.
   5. Tighten the adjusting nut onto the tilt lever.
   6. Place the tilt lever in position and torque the set bolt.
      **Torque: 340 kg-cm (25 ft-lb, 33 N·m)**
      > **NOTE:** The bolt has LH threads. Do not move the tilt lever before installing the steering column.
6. Install the tension springs and grommets.
   1. Install the bolt with spring to the bracket.
   2. Install the grommet into the bracket.
   3. Hook the spring into the grommet.
7. Install the combination switch.
   1. Install the combination switch with four screws.
   2. Install the connector with two screws.
   3. Install the clamp.

<a id="p783"></a>
### Installation of steering column assembly

*(See Removal of steering column assembly, manual page SR-3.)*

1. Place the steering column assembly in the installed position.
   1. Place the steering column assembly with the column upper cover and temporarily
      tighten support nuts and bracket nuts.
   2. Position the steering column assembly so the ends of lower support holes and
      mounting bolts touch.
2. Torque the tilt steering support nuts and breakaway bracket nuts.
   **Torque: 260 kg-cm (19 ft-lb, 25 N·m)**
3. Install the pedal bracket stay bolt. Install and torque the bolt.
   **Torque: 185 kg-cm (13 ft-lb, 18 N·m)**
4. Connect the ignition switch connector and combination switch connector.
5. Install the universal joint.
   1. Align matchmarks on the main shaft, universal joint and pinion shaft.
   2. Install and torque the two bolts.
      **Torque: 360 kg-cm (26 ft-lb, 35 N·m)**
6. Install the steering wheel.
   1. Align matchmarks on the main shaft and steering wheel, and install them.
   2. Check the steering wheel center point.
   3. Install and torque the steering wheel nut.
      **Torque: 350 kg-cm (25 ft-lb, 34 N·m)**
   4. Connect the horn connector and push in the wheel pad.
   5. Tighten the screw.
7. Connect the negative cable to the battery.

## Steering Gear Housing

<a id="p785"></a>
### Components

*(Figure: exploded view of the rack-and-pinion steering gear housing — see page image p.787, manual page SR-16.)*

Tightening torques called out on the components and removal diagrams (manual pages SR-14 and SR-16):

| Fastener | Torque — kg-cm (ft-lb, N·m) |
| --- | --- |
| Universal joint bolts | 360 (26, 35) |
| Tie rod end-to-knuckle arm nut | 500 (36, 49) |
| Gear housing bracket / mounting bolts | 440 (32, 43) |
| Crossmember bolts | 260 (19, 25) |
| Pinion bearing adjusting screw lock nut | 1,150 (83, 113); with SST: 930 (67, 91) |
| Rack guide spring cap lock nut | 700 (51, 69); with SST: 570 (41, 56) |
| Rack end | 850 (61, 83); with SST: 730 (53, 72) |
| Tie rod end lock nut | 480 (35, 47) |

> **NOTE:** In the diagram legend, values marked "•" are the torque to use when the fastener
> is tightened *through the SST* (i.e. with the specified torque-wrench fulcrum length); the
> unmarked value is the actual specified torque on the fastener.

- Non-reusable parts (per components diagram): oil seal, claw washer.
- Precoated parts (per components diagram): adjusting screw lock nut, adjusting screw, cap, cap lock nut.

<a id="p785"></a>
### Removal of gear housing

*(Figure: exploded view — see page image p.785, manual page SR-14.)*

1. Remove the universal joint.
   1. Place matchmarks on the main shaft, joint yoke and pinion shaft.
   2. Remove the two bolts.
   3. First pull the universal joint from the gear housing. Then pull it out from the main shaft.
2. Remove the front wheel.
3. Disconnect the tie rod ends.
   1. Remove the cotter pin and nut holding the knuckle arm to the tie rod.
   2. Using SST, disconnect the knuckle arm from the tie rod end. **SST 09628-62011**
4. Remove the center floor crossmember.
5. Remove the gear housing assembly.
   1. Remove the gear housing bracket set bolts.
   2. Remove the gear housing assembly.

### Disassembly of gear housing

1. Clamp the gear housing in a vise. Using SST, clamp the gear housing in the vise. **SST 09612-00012**
2. Remove the tie rods.
   1. Loosen the lock nut and place matchmarks on the tie rod and rack end.
   2. Remove the tie rod and lock nut.
3. Remove the clips, clamps and rack boots.
4. Remove the rack ends and claw washers.
   1. Unstake the claw washer.
      > **CAUTION:** Avoid any impact to the rack.
   2. Using SST, remove the rack ends. **SST 09612-10093 (09628-10020), 09612-24013 (09617-24010)**
      > **NOTE:** Mark the left and right rack ends.
   3. Remove the claw washer.
5. Remove the rack guide spring cap lock nut. Using SST, remove the rack guide spring cap
   lock nut. **SST 09612-10093 (09617-10010)**
6. Remove the rack guide spring cap. Using SST, remove the rack guide spring cap. **SST 09612-24013 (09612-10022)**
7. Remove the rack guide spring.
8. Remove the rack guide and rack guide seat.
9. Remove the pinion bearing adjusting screw lock nut. Using SST, remove the pinion
   bearing adjusting screw lock nut. **SST 09612-10093 (09617-10010)**
10. Remove the pinion bearing adjusting screw. Using SST, remove the pinion bearing
    adjusting screw. **SST 09612-24013 (09616-10020)**
11. Remove the pinion with upper bearing.
    > **NOTE:** Be careful not to damage the serrations.
    1. Fully pull the rack from the housing side and align the rack notched portion with the pinion.
    2. Remove the pinion together with the upper bearing.
12. Remove the rack. Remove the rack from the pinion side without revolving it.
    > **NOTE:** If the rack is pulled from the tube side, there is a possibility of damaging the bushing with the rack teeth surface.

<a id="p790"></a>
### Inspection and repair of gear housing components

1. Inspect the rack.
   1. Check the rack for runout and for teeth wear or damage.
      **Maximum runout: 0.3 mm (0.012 in.)**
   2. Check the back surface for wear or damage. If faulty, replace it.
      > **CAUTION:** Do not use a wire brush when cleaning.
2. Inspect the pinion bearings.
3. If necessary, replace the pinion upper bearing.
   1. Remove the upper bearing with SST. **SST 09950-20017**
   2. Install a new upper bearing with SST. **SST 09612-24013 (09612-10061)**
4. If necessary, replace the pinion lower bearing.
   1. Heat the rack housing to above 80°C (176°F).
   2. Tap the rack housing with a plastic hammer or such to remove the lower bearing by recoil.
   3. Heat the rack housing to above 80°C (176°F).
   4. Install a new lower bearing with SST. **SST 09620-30010 (09627-30010, 09631-00020)**
      > **NOTE:** Observe the correct bearing direction.
5. If necessary, replace the rack bushing.
   1. Push in the three claws of the bushing to the rack housing and pull out the bushing.
   2. Insure that the tube hole is not clogged with grease.
      > **NOTE:** If the tube hole is clogged, the pressure inside the boot will change after it is assembled and the handle turned.
   3. Align the rack housing hole to the three claws of the bushing and install it.
6. If necessary, replace the pinion oil seal.
   1. Using SST, drive out the oil seal. **SST 09620-30010 (09631-00020), 09630-24013 (09620-24010)**
   2. Using SST, drive in a new pinion oil seal until it is protruding 0.5 mm (0.020 in.).
      **SST 09620-30010 (09631-00020), 09630-24013 (09620-24020)**

<a id="p792"></a>
### Assembly of gear housing

*(See Components, manual page SR-16.)*

1. Pack molybdenum disulphide lithium base grease on the parts shown *(see page image p.792, manual page SR-21)*.
2. Install the rack into the rack housing.
   1. From the pinion side, install the rack into the rack housing.
   2. Set the rack notched side so that the pinion can be positioned inside.
   3. Line up the cutout portion of the rack with the pinion.
3. Install the pinion into the housing. Insure that the pinion end is securely in the lower bearing.
4. Install the pinion bearing adjusting screw.
   1. Coat liquid sealer onto the screw threads.
      **Sealant: Part No. 08833-00080, THREE BOND 1344, LOCTITE 242 or equivalent**
   2. Install the pinion bearing adjusting screw with SST. **SST 09612-24013 (09616-10020)**
5. Adjust the pinion preload.
   1. Line up the cutout portion of the rack with the pinion.
   2. Using SST, torque the pinion bearing adjusting screw to 3.7 kg-cm (3.2 in.-lb, 0.4 N·m).
      **SST 09612-24013 (09616-10010)**
   3. Using the same SST, loosen the pinion bearing adjusting screw to a torque of
      2.3 – 3.3 kg-cm (2.0 – 2.9 in.-lb, 0.2 – 0.3 N·m). **SST 09612-24013 (09616-10010, 09616-10020)**
      **Preload (torque): 2.3 – 3.3 kg-cm (2.0 – 2.9 in.-lb, 0.2 – 0.3 N·m)**
6. Install the pinion bearing adjusting screw lock nut.
   1. Coat liquid sealer onto the lock nut and housing contact surface.
      **Sealant: Part No. 08833-00080, THREE BOND 1344, LOCTITE 242 or equivalent**
   2. Install the lock nut and torque it with SST. **SST 09612-10093 (09617-10010), 09612-24013 (09616-10020)**
      **Torque: 930 kg-cm (67 ft-lb, 91 N·m)**
      > **NOTE:** Use a torque wrench with a fulcrum length of 42.5 cm (16.73 in.).
   3. Recheck the pinion preload. If incorrect, readjust.
      **Preload (turning): 2.3 – 3.3 kg-cm (2.0 – 2.9 in.-lb, 0.2 – 0.3 N·m)**
7. Install the rack guide seat and rack guide.
8. Install the rack guide spring.
9. Install the rack guide spring cap.
   1. Coat liquid sealer onto the guide spring cap threads.
      **Sealant: Part No. 08833-00080, THREE BOND 1344, LOCTITE 242 or equivalent**
   2. Mesh the rack with the pinion.
   3. Install the rack guide spring cap with SST. **SST 09612-24013 (09612-10022)**
10. Adjust the total preload.
    1. Tighten the rack guide spring cap and torque with SST. **SST 09612-24013 (09612-10022)**
       **Torque: 250 kg-cm (18 ft-lb, 25 N·m)**
    2. Using SST, return the rack spring cap 25°. **SST 09612-24013 (09612-10022)**
    3. Measure the total preload with SST. **SST 09612-24013 (09616-10010)**
       **Preload (turning): 5 – 13 kg-cm (4.3 – 11.3 in.-lb, 0.5 – 1.3 N·m)**
    4. If preload is insufficient: retorque the rack guide spring cap, and return it 12° or slightly less.
    5. If there is excess preload: slightly return the rack guide spring cap.
11. Install the rack guide spring cap lock nut.
    1. Coat liquid sealer onto the lock nut thread and housing surface.
       **Sealant: Part No. 08833-00080, THREE BOND 1344, LOCTITE 242 or equivalent**
    2. Tighten the lock nut and torque it with SST. **SST 09612-10093 (09617-10010), 09612-24013 (09612-10022)**
       **Torque: 570 kg-cm (41 ft-lb, 56 N·m)**
       > **NOTE:** Use a torque wrench with a fulcrum length of 34 cm (13.39 in.).
    3. Recheck the total preload. If incorrect, readjust.
       **Preload (turning): 5 – 13 kg-cm (4.3 – 11.3 in.-lb, 0.5 – 1.3 N·m)**
12. Install new claw washers and rack ends.
    1. Install a new claw washer.
       > **NOTE:** Align the claw of the claw washer with the rack groove.
    2. Install the rack end and torque it with SST. **SST 09612-10093 (09628-10020), 09612-24013 (09617-24010)**
       > **NOTE:** Use a torque wrench with a fulcrum length of 34 cm (13.39 in.).
       **Torque: 730 kg-cm (53 ft-lb, 72 N·m)**
    3. Stake the claw washer.
13. Install the rack boots.
    1. Install the rack boots.
       > **NOTE:** Be careful not to damage or twist the boots. The left and right boots are different. Be careful not to interchange them.
    2. Install the clamps and clips.
       > **NOTE:** Face the open ends outward, as shown, to avoid damage to the boots.
14. Install the tie rods.
    1. Screw the lock nuts and tie rods onto the rack ends until the matchmarks are aligned.
    2. After adjusting toe-in, torque the lock nuts.
       **Torque: 480 kg-cm (35 ft-lb, 47 N·m)**

<a id="p796"></a>
### Installation of gear housing

*(See Removal of gear housing, manual page SR-14.)*

1. Install the gear housing assembly. Install the four bolts and torque them.
   > **NOTE:** Be careful not to damage the boots. Align the centers of the column hole cover shield and the hole in the body where the pinion shaft passes through.

   **Torque: 440 kg-cm (32 ft-lb, 43 N·m)**
2. Install the center floor crossmember. Install the center floor crossmember and four bolts.
   **Torque: 260 kg-cm (19 ft-lb, 25 N·m)**
3. Connect the tie rod ends to the knuckle arms.
   1. Install the tie rod ends to the knuckle arms and torque the nuts.
      **Torque: 500 kg-cm (36 ft-lb, 49 N·m)**
   2. Install a new cotter pin.
4. Install the universal joint.
   1. Align matchmarks on the main shaft, universal joint and pinion shaft.
   2. Install and torque the two bolts.
      **Torque: 360 kg-cm (26 ft-lb, 35 N·m)**
5. Install the front wheels.
6. Adjust toe-in (see page FA-5).
7. Torque the tie rod end lock nuts.
   **Torque: 480 kg-cm (35 ft-lb, 47 N·m)**
8. Check the steering wheel center point.
9. Check the steering wheel free play.
